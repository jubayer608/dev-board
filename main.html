Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML:Returns or sets the HTML inside an element, including all tags.

innerText:Returns or sets only the visible text inside an element.

innerText:Returns or sets all text inside an element.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where a single event listener is added to a parent element instead of multiple child elements. It listens for events that bubble up from child elements and handles them efficiently.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a mechanism in the DOM event propagation process where an event starts from the target element and bubbles up to its ancestor until it reaches the document object.

Question-5: How do you create, add, and remove elements using JavaScript?

Create element: document.createElement('tag name')

Add element: parent.appendChild(variable name)

Remove element:parent.removeChild(child name)
